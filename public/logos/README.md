# Logo Assets Directory

Dieses Verzeichnis enthält die Logo-Bilder für das Logo-Carousel.

## Verwendung

1. Füge Logo-Bilder im SVG-Format hinzu (empfohlen für beste Qualität)
2. Benenne die Dateien nach dem Firmennamen (z.B. `salesforce.svg`, `hubspot.svg`)
3. Die Logos werden automatisch im Logo-Carousel angezeigt

## Empfohlene Spezifikationen

- **Format:** SVG (empfohlen) oder PNG mit transparentem Hintergrund
- **Breite:** 120-200px (SVG sollte skalierbar sein)
- **Höhe:** 30-50px
- **Hintergrund:** Transparent
- **Farben:** Original-Farben (wird automatisch graustufen gefiltert)

## Fallback-Verhalten

Wenn ein Logo-Bild nicht gefunden wird, wird automatisch ein SVG-Fallback verwendet. Dies stellt sicher, dass das Carousel immer funktioniert, auch wenn nicht alle Logos vorhanden sind.
